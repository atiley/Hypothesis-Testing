# Hypothesis-Testing
A sample of Python hypothesis testing using basic EDA, data cleaning and manipulation, and non-parametric statistical testing

Since 2002, The average number of goals in women's FIFA International soccer matches are statiscally higher than the average number of goals in the men's FIFA matches.
Women's matches garner an average of 2.98 goals per match; men's matches average 2.51 goals. The results are statisically significan at the 10% significance level, with a p value of 0.005.
